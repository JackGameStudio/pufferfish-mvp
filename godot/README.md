# 弹弹河豚 MVP - Godot 4 白盒测试版

## 如何运行

1. 打开 Godot 4（桌面上的 Godot.exe）
2. 点击 "Import" → 选择 `pufferfish-mvp/godot/project.godot`
3. 点击 "Run"（▶ 按钮）或按 F5

## 测试步骤

1. **启动游戏** → 应该看到蓝色圆形（鱼）在左侧，红色圆形（障碍物）分布在关卡中
2. **按住鼠标/空格** → 鱼开始膨胀 + 旋转
3. **松开** → 鱼沿反方向弹射，靠摩擦减速
4. **碰到红色圆形** → 扣血（HP 条在左上角）
5. **碰到灰色矩形** → 只推开，不扣血
6. **到达金色区域** → 过关
7. **HP 归零** → 游戏结束

## 已实现功能

- ✅ 充气膨胀（按住）
- ✅ 持续旋转（charging 时）
- ✅ 弹弓弹射（rotation + PI 方向）
- ✅ 摩擦减速飞行
- ✅ 圆形障碍（扣血）
- ✅ 矩形障碍（不扣血）
- ✅ 摄像机跟随
- ✅ HP 条 UI
- ✅ 胜利/失败判定
- ✅ 边界反弹
- ✅ 关卡数据（原版 28 圆形 + 16 矩形）

## 控制

- **鼠标左键 / 空格** → 充气 + 旋转，松开弹射
