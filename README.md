# 弹弹河豚 — 关卡编辑器

## 使用方法

1. 用浏览器打开 `editor.html`
2. 点击工具栏放置 **起点** → 放置 **终点** → 放置障碍物
3. 点击 ▶️ 试玩 测试关卡
4. 点击 📋 导出 复制 JS 代码，粘贴到游戏 `index.html`

## 快捷键

- `Ctrl+Z` 撤销 | `Ctrl+Y` 重做
- `Delete` / `Backspace` 删除选中
- 试玩中：`←/→` 调整角度 | `空格` 发射 | `ESC` 退出

## 拖拽调整世界尺寸

拖动画布**右边缘**调整宽度，拖动**下边缘**调整高度。

## 导出/导入

- **导出**: 点击 📋 导出，复制生成的 JS 代码到游戏 `index.html`
- **导入 JSON**: 点击 📂 加载，粘贴 JSON 即可加载关卡

## 测试关卡

导入 `editor-test-level.json` 快速体验。
