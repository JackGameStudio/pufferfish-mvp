# 弹弹河豚 MVP 🐡

一个简单的水下物理游戏，验证"膨胀旋转弹射"的核心玩法。

## 游戏介绍

控制一只河豚在水下环境中移动，通过膨胀和旋转来选择方向，然后弹射前进。

## 快速开始

1. 用浏览器打开 `index.html`
2. 按住空格键 → 河豚膨胀并旋转
3. 松开空格键 → 河豚向尾巴方向弹射
4. 到达蓝色终点区域过关

## 控制方式

| 操作 | 按键 | 说明 |
|------|------|------|
| 膨胀+旋转 | 空格（按住） | 河豚膨胀，同时自动旋转选择方向 |
| 弹射 | 空格（松开） | 向尾巴方向弹射，膨胀越大速度越快 |
| 重新开始 | 空格 | 过关或碰撞后按空格重新开始 |

## 游戏元素

| 元素 | 颜色 | 说明 |
|------|------|------|
| 起点 | 绿色圆 | 游戏开始的位置 |
| 终点 | 蓝色圆 | 到达这里过关 |
| 障碍物 | 红色圆 | 碰到会重置关卡 |
| 河豚 | 绿/黄/橙 | 绿色=正常，黄色=膨胀中，橙色=弹射中 |

## 核心参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 最大膨胀时间 | 1.0 秒 | 按住空格的最大时间 |
| 旋转速度 | 4 弧度/秒 | 约 230°/秒 |
| 弹射速度 | 700 px/s | 最大弹射速度 |
| 水的阻力 | 0.985 | 速度衰减系数 |
| 反弹系数 | 0.8 | 碰到边界反弹时的速度保留 |

## 物理特性

- **无重力环境**：模拟水下漂浮感
- **水的阻力**：弹射后速度会逐渐减小
- **碰撞反弹**：碰到边界会反弹并减速
- **速度阈值**：速度低于 5px/s 时自动停止
