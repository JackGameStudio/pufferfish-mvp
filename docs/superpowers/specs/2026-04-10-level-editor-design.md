# 关卡编辑器设计方案

- 日期：2026-04-10
- 状态：草稿
- 作者：麟虾（JACK 审批）

---

## 目标

为弹弹河豚 MVP 开发一个独立的关卡编辑器 `editor.html`，让 JACK 可以通过可视化界面设计关卡，无需手动编写坐标数据。

---

## 设计原则

- **独立文件**：编辑器与游戏 `index.html` 完全解耦，不修改游戏代码
- **单文件零依赖**：所有代码在一个 HTML 文件中，无外部库引用
- **直觉操作**：以鼠标操作为核心，类似 Figma/画图工具的交互体验

---

## 界面布局

```
┌──────────────────────────────────────────────────────────┐
│ 工具栏: [选择][圆形][方块][起点][终点] | [撤销][重做] | [试玩][导出] │
├──────┬───────────────────────────────────────────────────┤
│      │                                                   │
│ 侧边 │              画布（网格背景）                    │
│ 工具 │        边缘可拖拽改变世界尺寸                     │
│      │                                                   │
├──────┴───────────────────────────────────────────────────┤
│ 属性面板: x=, y=, w=/r=, [删除]     鱼大小: [====●====] 1.0x│
└──────────────────────────────────────────────────────────┘
```

### 工具栏
- **工具按钮**（左侧）：选择、圆形、方块、起点、终点，图标+文字
- **历史操作**（中间）：撤销、重做
- **模式切换**（右侧）：试玩、导出

### 侧边工具条（可选，显示当前选中工具高亮）

### 画布区
- 网格背景（辅助对齐）
- 世界边界（蓝色边框，同游戏）
- 支持缩放和平移

### 底部属性面板
- 选中物体时显示：x, y, w, h / r，可编辑
- 鱼大小滑块：0.5x ~ 2.0x，等比缩放

## 交互设计

| 操作 | 行为 |
|------|------|
| 左键点击空白（放置模式） | 在该位置放置当前工具对应的物体 |
| 左键点击物体（选择模式） | 选中该物体，显示属性面板 |
| 左键拖拽物体 | 移动物体位置 |
| 左键拖拽画布空白 | 平移视图（整体移动） |
| 拖拽画布四边/四角 | 改变世界尺寸 |
| 滚轮 | 缩放画布（0.25x ~ 4x） |
| Delete / Backspace | 删除选中物体 |
| Ctrl+Z | 撤销 |
| Ctrl+Y / Ctrl+Shift+Z | 重做 |
| Escape | 退出试玩模式 / 取消选中 |

### 起点和终点
- 起点：只能放置 1 个，放第二个时替换第一个
- 终点：只能放置 1 个，放第二个时替换第一个

## 游戏参数编辑

### 鱼大小（等比缩放）
```
鱼大小: [====●====] 1.0x   范围: 0.5x ~ 2.0x
```
- 同时缩放：`fish.radius` 和 `fish.maxInflate`
- 预览时反映在鱼的渲染大小上

### 世界尺寸
- 默认：2200 × 750（当前关卡尺寸）
- 拖拽画布边缘可调整
- 实时显示宽高数值

## 试玩模式

### 进入试玩
点击"试玩"按钮 → 编辑器 UI 隐藏，画布切换为游戏模式

### 试玩行为
- 起点放置河豚
- 点击河豚膨胀，松开弹射（同游戏逻辑）
- 圆形障碍造成伤害
- 方块障碍不造成伤害（仅推开）
- 到达终点区域 → 显示"通关！"提示

### 退出试玩
按 Escape 或点击"退出"按钮 → 返回编辑模式，所有物体位置保留

## 导出功能

### 复制代码
点击"导出" → 弹出模态框：
```
┌─────────────────────────────────┐
│  复制关卡代码                    │
│  ┌───────────────────────────┐  │
│  // 复制这段代码到游戏         │  │
│  const level = { ... }        │  │
│  └───────────────────────────┘  │
│  [复制代码]  [下载JSON]  [关闭] │
└─────────────────────────────────┘
```

**复制代码**：生成可直接粘贴到 `index.html` 的 JS 对象：
```javascript
const level = {
  startX: 80, startY: 450,
  endX: 2050, endY: 350, endRadius: 40,
  worldWidth: 2200, worldHeight: 750,
  fishScale: 1.0,
  obstacles: [
    { x: 200, y: 250, radius: 45 },
    { x: 350, y: 350, w: 100, h: 20 },
    // ...
  ]
};
```

**下载 JSON**：导出为 `level.json`，可加载回编辑器继续编辑

### 加载 JSON
点击"加载" → 文件选择器 → 解析 JSON → 还原所有物体和参数

## 数据结构

### 编辑器状态对象
```javascript
const editorState = {
  // 世界
  worldWidth: 2200,
  worldHeight: 750,

  // 鱼参数
  fishScale: 1.0,

  // 物体列表
  objects: [
    { id: 'start', type: 'start', x: 80, y: 450 },
    { id: 'end',   type: 'end',   x: 2050, y: 350, radius: 40 },
    { id: 'c1',    type: 'circle', x: 200, y: 250, radius: 45 },
    { id: 'r1',    type: 'rect',   x: 350, y: 350, w: 100, h: 20 },
    // ...
  ],

  // 历史（撤销/重做）
  history: [],
  historyIndex: -1,

  // UI 状态
  mode: 'edit',          // 'edit' | 'play'
  selectedId: null,
  activeTool: 'select',  // 'select' | 'circle' | 'rect' | 'start' | 'end'
  dragState: null,
  viewOffset: { x: 0, y: 0 },
  viewScale: 1.0,
};
```

### 导出格式（复制到游戏）
```javascript
// 导出到游戏时，合并圆形和方块到 obstacles 数组
const level = {
  startX: editorState.objects.start.x,
  startY: editorState.objects.start.y,
  endX: editorState.objects.end.x,
  endY: editorState.objects.end.y,
  endRadius: editorState.objects.end.radius,
  // 圆形障碍
  obstacles: [
    ...editorState.objects.filter(o => o.type === 'circle'),
    // 方块加 type: 'rect'
    ...editorState.objects.filter(o => o.type === 'rect').map(o => ({
      type: 'rect', x: o.x, y: o.y, w: o.w, h: o.h
    }))
  ]
};
```

## 技术实现

### 渲染
- 使用 HTML5 Canvas 2D，与游戏共用相同渲染逻辑
- 独立一套渲染函数（不引用游戏代码）
- 分辨率自适应：画布填满窗口，坐标系为逻辑像素

### 试玩模式
- 在编辑器 canvas 上运行精简版游戏循环
- 复用 `checkCollision` 等逻辑函数（从游戏代码复制）
- 不依赖游戏文件，独立运行

### 文件
```
pufferfish-mvp/
  index.html          # 游戏
  editor.html         # 编辑器（新）
```

### 依赖
- 零外部依赖
- 单个 HTML 文件，包含所有 CSS 和 JS
- 可直接在浏览器打开（file:// 协议亦可）

## 实现优先级

### Phase 1：MVP（最小可行版本）
- 画布渲染（网格+物体）
- 放置：圆形、方块、起点、终点
- 拖拽移动物体
- 撤销/重做
- 复制代码导出

### Phase 2：实用功能
- 试玩模式
- 加载/保存 JSON
- 改变世界尺寸
- 属性面板编辑

### Phase 3：体验优化
- 鱼大小滑块
- 物体缩放（拖拽控制点）
- 吸附网格
- 键盘快捷键

---

## 验收标准

1. 编辑器独立 HTML 文件，可直接在浏览器打开
2. 可放置圆形、方块、起点、终点并拖拽移动
3. 撤销/重做正常工作
4. "复制代码"按钮生成可粘贴到游戏的 JS 代码
5. 试玩模式可运行，能到达终点
6. 不修改 `index.html`，与游戏完全解耦
